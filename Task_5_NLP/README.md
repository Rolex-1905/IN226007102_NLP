## Overview
This project focuses on fine-tuning a BERT/DistilBERT model for sequence labeling tasks like Part-of-Speech (POS) Tagging and Chunking. The model predicts grammatical and phrase-level tags for each word in a sentence.

## Dataset

## Model

---
