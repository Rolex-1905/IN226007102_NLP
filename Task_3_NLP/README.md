# NLP Chatbot using Hugging Face Transformers

## Project Overview
This project is part of the Data Science Internship (February 2026) – NLP Task 3.

The objective is to build a conversational chatbot using a pre-trained transformer model from Hugging Face. The chatbot interacts with users in natural language and generates meaningful responses dynamically.

---

## Objectives
- Build a chatbot using transformer-based models
- Understand how pre-trained NLP models work
- Implement prompt-based text generation
- Create an interactive conversational system

---

## Technologies Used
- Python
- PyTorch
- Hugging Face Transformers
- Jupyter Notebook

---

## Model Used
- `microsoft/DialoGPT-medium` (pre-trained conversational model)

---

## Chatbot Workflow
User Input → Tokenization → Model Processing → Response Generation → Output Display → Loop until exit

---

## 💻 Features
- Interactive console-based chatbot
- Dynamic response generation (no hardcoding)
- Continuous conversation handling
- Exit condition (`exit` / `quit`)
- Uses state-of-the-art transformer model
